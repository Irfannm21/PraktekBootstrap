Form Forms adalah komponen Bootstrap untuk membuat... form. Atau lebih tepatnya mempercantik tampilan form HTML.
Form Polos

Form Control .form-control. Class ini memberikan style umum kepada inputan form, diantaranya tag input yang berbentuk textbox seperti input type="text", input type="password", input type="email, select> dan textarea

Form Group .form-control memang sudah menarik, tapi masih ada yang terasa kurang pas, yakni jarak antara inputan form dengan teks label di bawahnya terlalu rapat. Ini sudah di antisipasi Bootstrap dengan menyediakan class .form-group. Namun class ini tidak untuk inputan form, tapi sebagai container dari pasangan tag <label> dan tag <input> dengan kata lain di bungkus div.

Form Size Untuk membuat inputan yang lebih kecil atau lebih besar, tersedia class .form-control-sm dan form-control-lg

Form Readonly, Disabled dan plaintext Jika ditambahkan atribut readonly, inputan form menjadi tidak bisa diakses namun tampilannya masih normal. Sedangkan jika menggunakan atribut disabled, selain tidak bisa diakses tampilan form juga menjadi abu-abu yang menandakan tidak aktif.

Form File dan Form Range Untuk men-style inputan form yang berbentuk tag input type="file" dan input type="range", tersedia class khusus .form-control-file dan .form-control-range





Form Checkbox dan Radio Penggunaan atribut for di dalam tag <label> yang berpasangan dengan atribut id di tag <label>, menambah fitur user experience ke dalam checkbox (dan juga nantinya radio). Kita bisa men-checklist checkbox hanya dengan men-klik teks saja.

Form Grid Dari contoh form Bootstrap yang sudah kita coba, terlihat bahwa tambahan class .form- control membuat tampilan inputan form memanjang memenuhi parent element (width:100%). Namun jika digabung dengan Bootstrap grid system, kita bisa dengan mudah mengatur lebar dan posisi inputan ini.

col. Apabila anda masih ingat, class .col tanpa jumlah segmen akan membagi .row dengan ukuran yang sama besar.

Horizontal Form Dalam contoh sebelumnya, posisi tampilan tag <input> berada di bawah tag <input>. Bagaimana jika kita ingin membuat inputan form yang tampil bersebelahan dengan teks label secara horizontal (bukan di bawahnya)

Horizontal Form Label Size Untuk form yang ditempatkan secara horizontal, terdapat class khusus .col-form-label-sm dan .col-form-label-lg
