Alert Dismissing dalah sebutan untuk efek interaktif alert dengan tambahan icon close. Icon ini bisa di klik untuk menutup tampilan alert.

Irfans class .alert-dismissible, .fade dan .show (baris 1). Icon close sendiri dibuat dari tag <button> dengan class .close serta atribut data- dismiss="alert" (baris 3) . Isi tag <button> berupa tag <span>×<span> (baris 4). Kode × adalah kode HTML entities untuk menampilkan tanda silang ( x ).

Modal odal adalah sebutan untuk jendela atau tampilan yang berada di atas jendela lain Biasanya jendela modal ini tampil untuk memberikan informasi, peringatan atau untuk meminta inputan dari user.

.modal.fade#contohModal tabindex="-1">.modal-dialog>.modal-content>.modal-body>p ambahan atribut tabindex="-1" berguna untuk mencegah nagivasi dengan tombol tab ketika modal ditampilkan.

Agar teks di dalam modal tampil lebih rapi, bisa ditambahkan class .modal-title dan .modal-footer yang mirip seperti di komponen card:

.modal-dialog>.modal-content>.modal-header>h5.modal-title

.modal-dialog>modal.content>modal-body>p

Modal Centered Sebagai alternatif terdapat class .modal-dialog-centered yang bisa ditambahkan agar box modal tampil persis di tengah- tengah web browser.

.modal-dialog>.modal-content>.modal-header>h5.modal-title

.modal-dialog>.modal-content>.modal-body>p

Modal Size erdapat 3 pilihan ukuran box modal: small, regular dan large. Lebar box yang sudah kita pakai dalam contoh sebelumnya adalah ukuran regular yang merupakan lebar default. Untuk mengecilkan ukuran box modal menjadi small, tambahkan class .modal-sm ke dalam tag <div> yang memiliki class .modal-dialog. Untuk ukuran large, tambahkan class .modal-lg

Popovers Popovers (bentuk jamak dari popover) adalah komponen Bootstrap untuk membuat efek teks yang tampil di samping element saat ini. Popover mirip seperti tooltips namun dengan tampilan yang lebih menarik. Popover sendiri berasal dari library JavaScript Popper.js.

POPOVER DISMISS Alternatif yang lebih baik adalah seperti komponen modal, dimana kita bisa menutup modal cukup dengan men-klik sembarang tempat di luar element modal. Untuk membuat efek yang sama di popover, tambahkan atribut tabindex="0" dan data-trigger="focus"

Popover Trigger Pada dasarnya, hampir semua element HTML bisa digunakan untuk menampilkan popover (sebagai trigger), dengan syarat kita menulis semua atribut yang diperlukan.
