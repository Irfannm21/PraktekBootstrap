Pagination hadir dengan 3 ukuran, yakni (seperti yang bisa anda tebak): large, regular dan small. Untuk membuat ukuran large, gunakan class pagination-lg, dan untuk ukuran small, gunakan class pagination-lg.

Efek active dan disabled sangat cocok dipakai ke dalam pagination. Class .activemenandakan bahwa user sedang berada di halaman tersebut. Sedangkan class .disabled bisa dipakai untuk menandakan bahwa user berada di halaman terakhir, dimana tombol "Next" sudah tidak bisa di klik lagi

Secara default, pagination tampil di sisi kiri halaman. Untuk mengubahnya agar tampil di tengah halaman, tambahkan class .justify-content-center. Dan untuk mengaturnya agar tampil di sisi kanan halaman, tambahkan class .justify-content-end.

Badges adalah komponen Bootstrap yang berbentuk label dengan tampilan mirip seperti button. Badges bisa dibuat dari tag <span> maupun tag <a>.
